# Universus — Plataforma de Quizzes Interativa

## Sobre o Projeto

O **Universus** é uma plataforma web interativa para criação, gerenciamento e execução de quizzes, desenvolvida com o objetivo de facilitar a aplicação de avaliações e atividades educacionais de forma simples e dinâmica.

A aplicação possui fluxos diferentes para **professores** e **alunos**, permitindo que educadores criem quizzes personalizados e que participantes respondam utilizando um sistema de acesso por código.

O projeto foi desenvolvido como parte de uma iniciativa acadêmica, aplicando conceitos de desenvolvimento **frontend**, **backend**, **banco de dados relacional** e **autenticação de usuários**.

---

## Funcionalidades

### Professores

* Criação e gerenciamento de quizzes
* Cadastro de perguntas e alternativas
* Organização de conteúdos avaliativos
* Acompanhamento dos resultados dos participantes

### Alunos

* Entrada em quizzes através de código de acesso
* Execução das perguntas em tempo real
* Registro das respostas
* Visualização de resultados

### Frontend

* HTML

* JavaScript

### Backend / Banco de Dados

* Supabase
* PostgreSQL

### Ferramentas

* Git e GitHub
* APIs
* Controle de versão

---

## Principais Implementações

* Sistema de autenticação de usuários utilizando Supabase
* Persistência de dados em banco relacional PostgreSQL
* Modelagem de entidades como usuários, quizzes, perguntas e respostas
* Integração entre frontend e backend utilizando JavaScript
* Sistema de acesso por código para execução dos quizzes
* Estruturação de diferentes permissões e fluxos para professores e alunos

---

## Objetivo

O projeto busca oferecer uma alternativa simples e acessível para criação de avaliações digitais, reduzindo a complexidade de ferramentas tradicionais e proporcionando uma experiência mais interativa para professores e alunos.
